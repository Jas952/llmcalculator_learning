# Go Calculator. Updated version from Ch5 "Go Idioms" by Jon Bodner

This program is used as a practice tool for working with the Golang language. It is based on an example provided in the supplementary literature.

## Features

- Solving mathematical expressions
- Entering expressions directly through the terminal
